# 🎓 CS Self-Study Path

> A structured CS curriculum for working professionals — free resources, atomic sessions, real projects.

---

## Who this is for

- You work in software or data and want to fill the gaps in your foundations
- You want a solid CS foundation, not just tutorials
- You have limited time but long-term commitment
- You want to use **free resources only**

---

## The Guiding Frameworks

This curriculum follows the order and philosophy of:

- **[teachyourselfcs.com](https://teachyourselfcs.com)** — the best single-page CS curriculum on the internet
- **[OSSU Computer Science](https://github.com/ossu/computer-science)** — free, open-source degree equivalent

Both agree on the same core subjects. This path adapts them to real-world time constraints.

## The Learning Path at a Glance

| Phase | Topic | Primary Resource |
|-------|-------|-----------------|
| 0 | Math Foundations | Fleck's Discrete Math (free online) |
| 1 | Computer Architecture | cpu.land + nand2tetris |
| 2 | Systems Programming in C | Beej's Guide to C + build kilo editor |
| 3 | Algorithms & Data Structures | VisuAlgo + Classic CS Problems |
| 4 | Tools & UNIX | Art of Command Line |
| 5 | PLT & Compilers | Crafting Interpreters |
| 6 | Networking | Beej's Network Programming |
| 7 | Databases | cstack's SQLite tutorial |
| 8 | AI / Machine Learning | udlbook + Karpathy videos |
| 9 | Software Engineering | System Design Primer |
| 10 | Security & RE | OpenSecurityTraining2 |
| S | Data Engineering | DE Zoomcamp |

## Time Budget

| Slot | Time | Activity |
|------|------|----------|
| Weekday commute/lunch | 30 min | Phone reading (text-based) |
| Weekday evening | 20 min | Short YouTube video (≤20 min) |
| Weekend session | 1–2 hrs | Project build (hands-on) |

> 💡 **Key insight:** One chapter of Beej's Guide read on your phone beats zero CS50 lectures watched this month.
