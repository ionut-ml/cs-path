# Advanced Topic — Machine Learning

## Prerequisites

**Do NOT start without:**
- Phase 0 (Math) — linear algebra and probability fully covered
- Phase 4 (Algorithms) — you need Big-O intuition
- Phase 8 (Databases) — understanding data at scale

If you skip the prerequisites, you'll memorize APIs without understanding anything.
